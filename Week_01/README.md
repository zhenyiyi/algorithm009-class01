
### 解题步骤
1. 遇到题目第一先通过暴力求解
2. 看一下暴力求解是否可以优化部分代码 减少时间复杂度， 比如空间换时间
3. 总结下是否有哪些规律，思想升维
4. 养成自顶向下的编程习惯
